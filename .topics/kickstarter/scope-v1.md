# Scope v1

## What This Kickstarter Funds

This Kickstarter funds continued exploration and development of:
- explicit continuity artifacts
- `.trace.md` workflows
- recoverable lineage structures
- public documentation
- tooling experiments
- export/import workflows
- continuity inspection utilities
- operational research
- public educational material

---

## What This Project Is Trying To Achieve

The project explores infrastructure for:
- recoverable AI-assisted workflows
- inspectable continuity
- portable collaboration
- lineage-aware systems
- runtime-agnostic continuity structures

---

## What This Project Is NOT

This project does NOT attempt to:
- replace Git
- build AGI
- create a blockchain
- create surveillance systems
- create permanent identity systems
- lock users into proprietary platforms
- automate humans out of decision-making
- pretend all outputs are objectively true

---

## Practical Focus

The practical focus is:
- continuity
- reconstruction
- lineage
- inspectability
- recoverability
- explicit state handling

---

## Design Constraints

The ecosystem intentionally prefers:
- simple artifacts
- human-readable formats
- recoverable structures
- inspectable systems
- explicit uncertainty
- operational realism

Complexity should only be introduced when it provides measurable value.
