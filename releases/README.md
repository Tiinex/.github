# Tiinex Releases And Status

This folder is the stable home for transient Tiinex release notes, implementation status, and public progress snapshots.

It is not the main identity surface for Tiinex.

Use the files at the repo root for stable orientation:

- `profile/README.md` for the human-first org entrypoint
- `tiinex.orientation.v1.md` for stable Tiinex identity
- `tiinex.context.v1.md` for self-contained handoff and LLM context
- `tiinex.orientation.manifest.v1.json` for the machine-readable file map

Release files under this folder should answer questions like:

- What changed recently?
- What is implemented now?
- What is still provisional?
- What public status should readers understand today?

They should not redefine what Tiinex is.
